# Multi Vendor E-Commerce Backend

## Features
- JWT Authentication
- Role Based Authorization
- Vendor Management
- Product CRUD
- Category CRUD
- Cloudinary Image Upload
- Cart
- Wishlist
- Address
- Orders
- Razorpay Payment
- Reviews & Ratings
- Dashboard Analytics
- Search, Filter & Pagination
- Coupon System

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cloudinary
- Razorpay

## Installation

npm install

## Environment Variables

PORT=
MONGODB_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=

## Run

npm run dev
